DROPDOWN: 
dropdown mainly functions with the select tag.
Selects one or multiple options in the <select> element with locator.selectOption(). 
You can specify option value, or label to select. Multiple options can be selected.

// Single selection : visible text
await page.locator(xpath).selectOption('blue');

// Single selection : using label/visible text
await page.locator(xpath).selectOption({ label: 'Blue' });

// Single selection : using index
await page.locator(xpath).selectOption({ index: 4});

//another approach:
await page.selectOption(xpath,'visible text');
